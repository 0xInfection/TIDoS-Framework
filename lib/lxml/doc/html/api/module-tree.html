Module Hierarchy

  • lxml
    • lxml.ElementInclude: Limited XInclude support for the ElementTree package.
    • lxml._elementpath
    • lxml.builder: The E Element factory for generating XML documents.
    • lxml.cssselect: CSS Selectors based on XPath.
    • lxml.doctestcompare: lxml-based doctest output comparison.
    • lxml.etree: The lxml.etree module implements the extended ElementTree API for XML.
    • lxml.html: The lxml.html tool set for HTML handling.
      • lxml.html.ElementSoup: Legacy interface to the BeautifulSoup HTML parser.
      • lxml.html.builder: A set of HTML generator tags for building HTML documents.
      • lxml.html.clean: A cleanup tool for HTML.
      • lxml.html.defs
      • lxml.html.diff
      • lxml.html.formfill
      • lxml.html.html5parser: An interface to html5lib that mimics the lxml.html interface.
      • lxml.html.soupparser: External interface to the BeautifulSoup HTML parser.
    • lxml.includes
    • lxml.isoschematron: The lxml.isoschematron package implements ISO Schematron support on top of the pure-xslt 'skeleton' implementation.
    • lxml.objectify: The lxml.objectify module implements a Python object API for XML. It is based on lxml.etree.
    • lxml.pyclasslookup
    • lxml.sax: SAX-based adapter to copy trees from/to the Python standard library.
    • lxml.tests: The lxml test suite for lxml, ElementTree and cElementTree.
      • lxml.tests.common_imports
      • lxml.tests.dummy_http_server: Simple HTTP request dumper for tests in Python 2.5+.
      • lxml.tests.selftest
      • lxml.tests.selftest2
      • lxml.tests.test_builder
      • lxml.tests.test_classlookup: Tests for different Element class lookup mechanisms.
      • lxml.tests.test_css
      • lxml.tests.test_doctestcompare
      • lxml.tests.test_dtd: Test cases related to DTD parsing and validation
      • lxml.tests.test_elementpath: Tests for the ElementPath implementation.
      • lxml.tests.test_elementtree: Tests for the ElementTree API
      • lxml.tests.test_errors
      • lxml.tests.test_etree: Tests specific to the extended etree API
      • lxml.tests.test_external_document: Test cases related to direct loading of external libxml2 documents
      • lxml.tests.test_htmlparser: HTML parser test cases for etree
      • lxml.tests.test_http_io: Web IO test cases that need Python 2.5+ (wsgiref)
      • lxml.tests.test_incremental_xmlfile: Tests for the incremental XML serialisation API.
      • lxml.tests.test_io: IO test cases that apply to both etree and ElementTree
      • lxml.tests.test_isoschematron: Test cases related to ISO-Schematron parsing and validation
      • lxml.tests.test_nsclasses: Test cases related to namespace implementation classes and the namespace registry mechanism
      • lxml.tests.test_objectify: Tests specific to the lxml.objectify API
      • lxml.tests.test_pyclasslookup: Tests specific to the Python based class lookup.
      • lxml.tests.test_relaxng: Test cases related to RelaxNG parsing and validation
      • lxml.tests.test_sax: Test cases related to SAX I/O
      • lxml.tests.test_schematron: Test cases related to Schematron parsing and validation
      • lxml.tests.test_threading: Tests for thread usage in lxml.etree.
      • lxml.tests.test_unicode
      • lxml.tests.test_xmlschema: Test cases related to XML Schema parsing and validation
      • lxml.tests.test_xpathevaluator: Test cases related to XPath evaluation and the XPath class
      • lxml.tests.test_xslt: Test cases related to XSLT processing
